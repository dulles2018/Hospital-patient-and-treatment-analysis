# Hospital and Patient Treatment Analysis (SQLite)

## Overview
This project analyzes hospital patient data using SQL in SQLite.

## Skills Demonstrated
- SQL joins
- Aggregations
- Grouping and filtering
- Healthcare data modeling

## Key Questions Answered
- What is the average treatment cost?
- Which doctor handles the most patients?
- What are the most common diagnoses?
- What is the recovery rate?

## Tools Used
- SQLite
- SQL

## How to Run
1. Open SQLite
2. Run schema.sql
3. Run data.sql
4. Run queries.sql
